Greyscale image coloration



This project consisted in creating models to predict the colors of greyscale images following the CIELAB color scheme. Each greyscaled image represented the L parameter and our models had to predict the a and b channels that represent colors between green and red and blue and yellow respectively. We trained multiple models using U-Net and GAN architectures and compared their outputs given differents training parameters.

Python, PyTorch

Base game engine creation


In this project, I created a basic game engine from scratch in Python. This engine allowed for : - Perspective and orthographic projections - Wireframe and solid display - Translation, rotation and scaling using transformation matrices. - The rotations were displayed under different forms: Euler angles, Exponential map and Quaternions - LERP and SLERP to return to the initial state - Camera movement (translation, rotation) - Lighting using the diffuse lighting model - Backface culling to prevent artefacts

Python
